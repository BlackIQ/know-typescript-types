# TypeScript tutorial

## Step 1

Make a directory and install typescript inside it.

```bash
mkdir tutorial
cd tutorial
npm i typescript
```

## Step 2

You need a `tsconfig.json` file and this is how you get it.

```bash
npx tsc --init
```

## Step 3

Make a file and start writing code.

```bash
touch index.ts
```
